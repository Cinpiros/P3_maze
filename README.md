# P3 PierreT

## setup:

* python3 -m venv p3_pierreT
* source p3_pierreT/bin/activate
* py -m pip install -r requirements.txt
* py -m main.py

-----------------------------------------

## how to use:

this program is a game
where the goal is to collect the objects
in the labyrinth and go to the guard to win.
